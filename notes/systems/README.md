# Systems

The **system map** for this repo — the moving parts and how they fit. Start with
[`overview.md`](overview.md). Unlike a code project with many subsystems, this
repo has a small number of cooperating systems (the website, the blog, the hub,
the cross-project sync), so they're covered together in `overview.md` rather than
split into per-system files. Add a dedicated file here if one of them grows
enough to warrant it.
