# 2026-06-23 — Session Log

## "Midnight Archive" — overhaul the index pages' visual identity (0.4.0)

Acted on direct feedback that the site "looks like Bootstrap / a quick template" on
every page except the blog articles. The articles read well because they're just
well-set prose; the index pages (home, projects, downloads, docs) leaned on the
default templated vocabulary — rounded shadowed cards, filled pills, and stacked
equal-card grids — which is exactly what reads as "generated."

**Direction.** Gave the index pages a distinct **catalogue / ledger identity** that's
recognizable even with the logo removed, while keeping the existing violet/amber
palette, AA contrast, and dark-first art direction (per `context/principles.md`):

- **Hairlines, not shadows.** `--shadow:none`; structure comes from 1px rules over a
  deeper ink background. Softer radii, lightly squared buttons.
- **A monospace catalogue system.** Section kickers, project statuses, tags, dates,
  nav group titles, the footer, and "more →" links are JetBrains Mono with letter-
  spacing and tabular numerals — the technical-archive register.
- **Home rebuilt for rhythm** (`index.html`): hero (editorial headline + lead + a
  bordered stat ledger) replacing the compact masthead → **Current focus** feature for
  the active project (picked by `status contains "Active"`, two-column with a "recent
  activity" log filtered from its tagged posts) → the other projects as **numbered
  rows** (`01`/`02` via CSS counters) → Latest updates as a **lead + tight rows** (not
  six equal cards) → a paired **Documentation + latest-builds** band.
- **Projects page** (`projects.md`): same numbered-row catalogue instead of the 3-up
  card grid.
- **Atmosphere** dialed back to one faint wash (no neon blobs).

**Mechanics.** All edits via the editor tools; build + git via PowerShell (the bash
mount is unreliable here). `assets/css/main.css` rewritten wholesale but **every
interior-page selector preserved** (`.article`/`.prose`/`.doc-*`/`.dl-*`/`.post-list`/
`.cta-strip`) so layouts didn't break. Local `JEKYLL_ENV=production jekyll build`
green; home, `/projects/`, `/downloads/`, `/docs/` spot-checked with a headless render
before shipping. MINOR bump `0.3.2 → 0.4.0`.
