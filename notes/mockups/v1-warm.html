Random AI Prompt

A controllable generator for random and dynamic text prompts, built around a dynamic-prompt language — with a CLI and a web UI.

Released v2.6.0 Active Updated Jun 22, 2026
JavaScriptNode 24Web UI

What it is

Random AI Prompt turns small, composable rules into rich, varied text prompts. Its dynamic-prompt language (DPL) makes generation controllable rather than purely random, and it runs as a CLI and a web app (a React + Vite SPA) over one shared core.
