Random AI Prompt turns small, composable rules into rich, varied text prompts. Its
**dynamic-prompt language (DPL)** makes generation controllable rather than purely random —
with OR-groups, weighting, and reusable lists. It runs as a CLI and a web app (a React +
Vite SPA) over one shared core, and ships with a Vitest + Playwright test suite and a
JSDoc doc-site.
