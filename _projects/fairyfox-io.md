This site. A hand-built Jekyll site (no external theme) that doubles as the **hub** for the
projects above: it holds the shared engineering standards, a living-notes system, and the
cross-project sync model — a git-only, on-request, non-recursive flow where the hub reads
the projects and the projects read the hub's standards. It deploys to GitHub Pages via
GitHub Actions on every push.
