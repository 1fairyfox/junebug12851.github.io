The original Pokered Save Editor — a complete, stable save-file editor for Pokémon Red
&amp; Blue, built with Electron, Angular, and TypeScript. It is no longer developed, but
remains the **recommended working tool** until the Qt 6 rewrite (Pokered Save Editor 2)
reaches parity. It sits outside the shared-standards mesh.
